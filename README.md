# McDonald-s-Menu-Nutritional-Analysis-Welcome to the McDonald's Nutritional Dataset Analysis. This project offers a comprehensive examination of the nutritional content across different food categories on McDonald's menu.This project was created to analyze the nutritional fact of multinational fast food chain McDonalds.McDonald's Menu, providing detailed nutritional information for each food category. By reviewing this analysis, individuals can make informed choices based on their preferences and dietary requirements.Dataset provides a list of all of the McDonalds menu items the nutritional value for these items.This analysis was performed using Jupiter Notebook, along with Python libraries like Pandas, NumPy, and Matplotlib for data handling and visualization.
